# 概述

## 简介

JSX Editor 提供多个 React 组件与 Hooks，读取 React 函数式组件代码， 通过画布进行所见即所得的可视化编辑并将结果转换为
React 代码。

提供一些插件，对于特定语法结构的代码展示相应的交互，如：

```jsx
// 展示条件渲染交互
state === 'welcome' ? <div>welcome</div> : <div>form</div>
// 展示列表交互
map((xs) => <li>{xs}</li>)(data)
```

提供变量监视器、故事设计器，用于设计人员描述不同场景下的 UI 组件。
